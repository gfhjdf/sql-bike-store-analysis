This project analyzes sales data from a bike store business using SQL Server.

The goal of the project was to transform raw transactional data into useful business insights and KPIs.

The analysis includes sales trends, product performance, inventory monitoring, staff performance evaluation, and investigation of revenue decline in 2018. This project analyzes sales data from a bike store business using SQL Server.

Investigation: Why did revenue drop in 2018?
Revenue dropped significantly in 2018 compared to previous years.

The analysis examined several possible causes:
- Discounts
- Product shortages
- Staff performance
